# eval-rubric-design

> Design structured evaluation rubrics for scoring LLM and agent outputs — quality dimensions, scoring scales, hard gates, and edge cases.

## Install

```bash
npx skills eval-rubric-design -a codex      # Codex
npx skills eval-rubric-design -a replit     # Replit
cp -r eval-rubric-design/ .agents/skills/   # Universal
```

## Usage

Trigger: "create an eval rubric", "define evaluation criteria", "design a rubric", "what should I evaluate"

## Platforms

Works with: Codex, Ampcode, Claude Code, Warp, Gemini CLI, GitHub Copilot, Cursor, VS Code, Replit, Factory.ai, Bolt.new
