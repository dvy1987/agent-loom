# eval-pipeline

> Design automated evaluation pipelines for LLM and agent systems — deterministic checks, statistical metrics, and LLM-as-judge scoring with CI/CD integration.

## Install

```bash
npx skills eval-pipeline -a codex      # Codex
npx skills eval-pipeline -a replit     # Replit
cp -r eval-pipeline/ .agents/skills/   # Universal
```
