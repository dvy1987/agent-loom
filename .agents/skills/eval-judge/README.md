# eval-judge

> Score LLM and agent outputs using LLM-as-judge — direct scoring against rubrics or pairwise comparison with built-in bias mitigation.

## Install

```bash
npx skills eval-judge -a codex      # Codex
npx skills eval-judge -a replit     # Replit
cp -r eval-judge/ .agents/skills/   # Universal
```

## Usage

Trigger: "score this output", "judge this response", "which response is better", "rate this", "pairwise comparison"

## Platforms

Works with: Codex, Ampcode, Claude Code, Warp, Gemini CLI, GitHub Copilot, Cursor, VS Code, Replit, Factory.ai, Bolt.new
